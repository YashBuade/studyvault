# Premium Logo Design for StudyVault

## Logo Concept

The StudyVault logo combines two powerful concepts:
1. **Modern Geometric Design** - Clean, minimal shapes that convey trust and sophistication
2. **Symbolic Meaning** - Represents secure knowledge storage and learning

## Visual Guidelines

### Primary Logo - Icon + Wordmark (Horizontal)

The logo consists of:
- **Icon**: A stylized geometric vault or shield shape
- **Wordmark**: "StudyVault" in premium sans-serif (Inter)
- **Layout**: Side-by-side, with balanced spacing

### Colors

**Light Mode:**
- **Primary Blue**: #3B82F6
- **Accent**: Gradient variants for depth

**Dark Mode:**
- **Light Blue**: #60A5FA  
- **Maintains premium feel with adjusted contrast

### Sizing

- **Large** (lg): 56x56px icon, 24px wordmark font
- **Medium** (md): 40x40px icon, 18px wordmark font
- **Small** (sm): 32x32px icon, 14px wordmark font

### Key Design Principles

1. **Minimal**: Uses only essential shapes
2. **Scalable**: Looks perfect at any size
3. **Memorable**: Simple enough to be iconic
4. **Versatile**: Works in light, dark, and monochrome
5. **Modern**: Contemporary geometric aesthetic
6. **Trustworthy**: Professional, corporate feel

## Technical Implementation

The logo should:
- Use SVG for perfect scaling
- Support CSS dark mode with `dark:` classes
- Include text and icon-only variants
- Have hover and focus states with transitions
- Be accessible with proper ARIA labels

## Brand Colors with Logo

When using the logo alongside content:
- Minimum clear space: 8px around icon
- Minimum size: 24x24px for web visibility
- Monochrome variant for constrained spaces
- White background not required (works on any background)

## Usage Context

- **Navigation/Branding**: Full logo with wordmark
- **Favicon**: Icon-only variant
- **Social Media**: Square format (1:1)
- **Documentation**: Primary logo (horizontal)
- **Blog/Posts**: Icon-only with attribution

## Premium Design Checklist

✓ Geometric and minimal
✓ Works at any scale
✓ Legible at small sizes
✓ Dark mode compatible
✓ Professional appearance
✓ Modern aesthetic
✓ Memorable and distinctive
✓ Accessible and inclusive
✓ Cultural sensitivity considered
✓ Timeless design (won't look dated)
